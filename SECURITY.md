# Security Policy

## Supported versions

We provide security updates for the latest `main` branch.

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Preferred: use GitHub Security Advisories:
- https://github.com/jason-allen-oneal/GhostMCP/security/advisories/new

If you cannot use GitHub advisories, open a minimal issue that does not include exploit details and ask for a private channel.
